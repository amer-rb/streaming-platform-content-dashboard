# Streaming Platform Content Dashboard

An interactive browser dashboard analyzing synthetic streaming-platform catalog data across platform, genre, release year, country, rating, and content-gap opportunities.

## Live Demo

https://streaming-platform-content-dashboard.netlify.app/

## What Is Included

- `index.html`: the dashboard page
- `styles.css`: visual styling
- `app.js`: chart and filter logic
- `data.js`: embedded project dataset
- `preview.png`: screenshot for portfolio thumbnails
- `.nojekyll`: keeps GitHub Pages from processing the site

## Portfolio Summary

Built an interactive streaming-content analytics dashboard with 28,732 synthetic titles across Netflix, Disney+, and Prime Video. The project includes KPI cards, filterable genre and country analysis, release trends, rating distribution, platform share, and a content-gap scoring panel to identify catalog opportunities.
