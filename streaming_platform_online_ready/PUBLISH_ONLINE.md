# Publish This Dashboard Online

## Fastest Option: Netlify Drop

1. Go to `https://app.netlify.com/drop`.
2. Drag the `streaming_platform_online_ready` folder into the drop area.
3. Netlify will give you a public URL.
4. Rename the site in Netlify if you want a cleaner link.

This is the easiest path because it does not require code, Git, or a build setup.

## Portfolio Option: GitHub Pages

1. Create a new public GitHub repository.
2. Upload the contents of this folder, not the folder itself.
3. Make sure `index.html` is at the root of the repository.
4. Go to repository `Settings`.
5. Open `Pages`.
6. Choose deploy from the `main` branch and the root folder.
7. Save and wait for GitHub to publish the site.

Your link will usually look like:

```text
https://YOUR-USERNAME.github.io/YOUR-REPOSITORY-NAME/
```
